Html Öğrenmeye Nasıl Başlanır?

Fatma Nur Aksakal | 05.07.2025

HTML (HyperText Markup Language): Web sayfalarının iskeletini oluşturan dildir.İnternette gördüğün yazılar, başlıklar, butonlar, görseller... hepsi HTML sayesinde yerleştirilir.

1.VS Code ya da Not Defteri ile Başla

Bir .html dosyası aç. (Örnek: index.html). Visual Studio Code kullanıyorsan çok rahat yazarsın.

2.Temel HTML Yapısını Öğren

Başlangıç kodu şöyledir: < !DOCTYPE html>

Temel bazı etiketler:

  • h1/h6:başlıklar
  • p:paragraf
  • img: görsel

3.Deneyerek ve Günlük Minik Alıştırmalarla Öğren

Kodları yaz, tarayıcıda (Chrome, Edge) aç. Ne olduğunu gözlerinle gör.Sağ tıklayıp "İncele" ile yapısını gözlemle.

Kendini tanıtan bir sayfa yapmayı deneyebilirsin.Basit örneklerle ilerle.
